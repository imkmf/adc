# ADC

Check the status of the Apple Developer Center from the command-line.

## Installation

Add this line to your application's Gemfile:

    gem 'adc'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install adc

## Usage

Run `adc` from the command line.
